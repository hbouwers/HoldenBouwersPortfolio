Lets Convention

Lets Convention is a similiar platform to Airbnb but specifically for convention goers. I worked on this full stack project adding many features such as messaging, user vs admin views, exporting calenders to sync with other sites, and many others. This was a MERN stack project (MongoDB, Express, React, Node) written mostly in Typescript. I really enjoyed this project and worked long hours figuring out how to make the message feature work well and integrating a google geo api. It is also worth noting that when I worked on this project I work on every part of it from front, back, to deployment with Docker/Kubernetes and GCP

Spark Client Manger

Spark Client manager is an ASP.NET MVC project build to manager freelance clients. The project uses C#/SQL and the web mvc model in visual studio. This app is meant to create a connection between the freelancer and the client. You can message the client, take notes on projects and share projects with other users
